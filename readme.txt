
README
——————————————————————————————————

This WebApp was made to collects tweets with a youtube link and #nowplaying hashtags, and show it to the users. Users can view the videos directly in the webapp or post a tweet to @BInowplaying account.


HELPERS
—————————————————
1. Boostrap v3.3.6

2. Jquery 1.8.2

3. Freegeoip -> Get City name - 

4. Kevin Liew - Twitter Timeline: http://bit.ly/1giCd64

5. A lot of Google Searchs 


REQUIRED
—————————————————
You have to Install a chrome extension to avoid errors on X-Frame-Options on the server. (godaddy.com)

Link: https://chrome.google.com/webstore/detail/ignore-x-frame-headers/gleekbfjekiniecknbkamfmkohkpodhe
